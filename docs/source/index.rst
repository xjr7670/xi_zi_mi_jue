自序
-----

尝读《笔阵图》谓藏之石室，千金勿传。鲁公得长史笔法，亦历时久远，疑作者故郑重其辞，欲后人勿轻视也。

吾先人书[缺]两世，媲美议论皆有成书。和生也晚，未见祖颜，先君子口授诸诀，稍知一二。奈质性鲁钝，学之廿年，功用屡迁。未臻其极。

和初习颜柳未成，即从辙董赵颇[X]时誉，自行君子见背后，始惧前业之将坠，奋志临礼泉铭、庙堂碑而法外之意，如有所得，惜乎不得趋庭以相证矣。

==========
钩法
==========

----------
长钩
----------

即趯也。法也详直画图内。此论出三角法。

.. image:: ../images/changgou.jpg
   :align: center

王虚舟先生论欧阳公书法点画俱示棱角，以便初学也。得法之后，去方就圆，亦易易耳。

勾凡三角，要在中间向右一宕，得了第一角；次向下弯，然后缩笔向上，左趯出连作三笔写出。
